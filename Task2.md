# Task 2: User Profile Management

## **Task Description**
Develop a complete module for managing user profiles, including viewing, updating, and saving changes.

## **Subtasks**
1. **Frontend**
   - Create a form for profile details (name, email, etc.).
   - Add form validation (e.g., email format, required fields).
   - Create a responsive design for profile view/edit pages.

2. **Backend**
   - Build API endpoints:
     - GET `/api/user/profile` - Fetch user profile details.
     - PUT `/api/user/profile` - Update user profile.
   - Add validation for input data.

3. **Database**
   - Add fields to `users` table for new profile attributes.
   - Write migration scripts for schema changes.
